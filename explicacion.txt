En este script se importa el modulo ProxmoxApi de la libreria proxmox para realizar la conexión con la API en primer lugar. Luego se importan los módulos random y string de la biblioteca estantar de Python. A continuación, se realiza la conexión con el servidor, a través de la ip pública (en este caso se usa el DDNS de softether aprobechando que esta creado del la VPN). Luego se generan un id y un nombre único para el contenedor. Por último se configuran los parámetros del contenedor en cuestión (template, memoria RAM, almacenamiento ...)
